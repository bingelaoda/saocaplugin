Creating a Repository Model

  1. In the Package Explor view, select the project. Then right-click on the project and select  New > Other... from the project´s context menu.

  2. Select Palladio Modeling > PCM Repository Model Diagram then click Next to start the wizard.

  3. On the next page click Finish to creat the PCM Repository Model Diagram.

  4. Right-click in the repository diagram view and select Load Resource...

  5. In the Load Resource wizard select Browse Workspace...

  6. Select your project and your resourcetype. Then click Ok.

  7. Click again Ok in the Load Resource wizard.
