Creating a System Model

  1. In the Package Explor view, select the project. Then right-click on the project and select  New > Other... from the project´s context menu.

  2. Select Palladio Modeling > system Diagram then click Next to open the New system Diagram wizard.

  3. On the next page click Finish to create the system diagram.

  4. Right-click in the system diagram view and select Load Resource...

  5. In the Load Resource wizard select Browse Workspace...

  6. Select your project and your repository. Then click Ok.

  7. Click again Ok in the Load Resource wizard.
