Provided Type

A provided type does not contain information about its internal realisation nor is it allowed to specify those informations. Hence all sub-types of a concrete provided are not limited concerning their internal realization and their required interfaces. The provided type is the component type that defined the fewest limitations for sub-types. Provided types aim at defining components - components whose primary aim is to be reused instead of making for example prescriptions for the realisation.
