Component Models

In recent years, many component models have been developed for many different purposes. A taxonomy of these models can be found in [2].
  • ROBOCOP: [42] allows performance predictions, aims at embedded systems.
  • PACC: [43] allows performance predictions, aims at embedded systems.
  • Koala: [] no QoS, aims at embedded systems
  • SOFA: [13] allows protocol checking.
  • Fractal: allows runtime reconfigurations of architectures
  • UML: [44] limited component concept
  • CCM: no QoS
  • EJB: no QoS
  • COM+: no QoS
