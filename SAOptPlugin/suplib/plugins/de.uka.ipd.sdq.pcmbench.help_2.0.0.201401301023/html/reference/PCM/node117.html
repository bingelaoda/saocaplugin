First Class Entities

First class entities are those entities of the PCM that can exist within model instances without depending on other entities. Modelling instances have to start with a first class entity. First class entities of the PCM are:
  • Components (all types)
  • Interfaces
  • Ressources
All other entities depend on first class entities - they cannot exist without an instance of first class entities. Those entities are called second class entities. For example a signature cannot exist without an interface.
