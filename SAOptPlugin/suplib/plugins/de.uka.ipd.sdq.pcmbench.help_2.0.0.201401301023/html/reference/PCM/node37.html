Definition

Mathematically, a random variable is defined as a measurable function from a probability space to some measurable space. More detailed, a random variable is a function

with = the set of observable events and being the set associated to the measurable space. Observable events in the context of software models can be for example response times of a service call, the execution of a branch, the number of loop iterations, or abstractions of the parameters, like their actual size or type. Note, that often a random variable has a certain unit (like seconds or number of bytes, etc.). It is important for the user of prediction methods to keep the units in the calculations and in the output to increase the understandability of the results.

A random variable is usually characterised by stochastical means. Besides statistical characterisations, like mean or standard deviation, a more detailed description is the probability distribution. A probability distribution yields the probability of taking a certain value. It is often abbreviated by . For discrete random variables, it can be specified by a probability mass function (PMF). For continuous variables, a probability density function (PDF) is needed. However, for non-standard PDFs it is hard to find a closed form (a formula describing the PDF). Because of this and for reasons of computational complexity, we use discretisized PDFs in our model.

For the event spaces we support include integer values , real values , boolean values and enumeration types (like "sorted" and "unsorted") for discrete variables and for continuous variables.
