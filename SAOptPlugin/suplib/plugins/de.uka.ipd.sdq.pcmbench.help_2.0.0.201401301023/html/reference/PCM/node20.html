Related Work

There are numerous publications on component-based development processes [,8,9,10,11,12]. However, most of these process descriptions do not deal with extra-functional properties. Furthermore, there are many approaches for the performance prediction of component-based software systems [6], but only few sketch the encompassing development process in detail or spread the needed information for QoS analyses among the participating roles.
