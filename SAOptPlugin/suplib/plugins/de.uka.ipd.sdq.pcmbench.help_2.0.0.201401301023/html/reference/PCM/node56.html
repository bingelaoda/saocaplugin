Implementation Component Type

fully specified components. provided interfaces (interfaces connected to a component by provided roles) define the services offered by a component to its environment required interfaces (interfaces connected to a component by required roles) specify the services a component needs in order to deliver its provided services.

The specification of the internal structure of a component depends on the realisation of the component. basic components: behavioural descriptions for each service in form of service effect specifications composite components: compositional structure, which components and their interconnection
