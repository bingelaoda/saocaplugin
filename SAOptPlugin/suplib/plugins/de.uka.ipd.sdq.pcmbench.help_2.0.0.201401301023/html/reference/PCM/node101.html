Implementation Type Occurrence

Basic and composite component have been mentioned before. Both are implementation types. In figure 3.4 they are shown with dashed lines as a layer below the component types discussed above. In contrast to the other component types which declare the outer properties of a component, basic and composite component add an abstraction of the inner structure and behaviour of a component. Basic components and composite components are the two kinds of component type realisations.
