Performance Meta-Models

To describe the performance properties of software systems, several meta-models have been introduced (Survey by [45]).
  • SPE-Metamodel: [46] designed for object-oriented software systems.
  • UML+SPT profile: [36] offers capabilities to extend UML models with performance properties. Is not suited for parametric dependencies, which are needed for component specifications.
  • CSM: [37] is closely aligned with the UML-SPT approach and does not target component-based systems.
  • KLAPER: [38] is designed for component-based architectures and reduces modelling complexity by treating components and resources in a unified way.
