Interpretations of Required Interfaces

We discuss three options for the semantics of required interfaces which we know about. Note that we distinguish them with numbers for further reference.

  1. A required interface can be used, but additional interfaces can be added.
  2. Only the listed required interfaces can be used and no more.
  3. A required interface has to be used in a predefined way and certain call sequences have to be executed.
